概述

除了在 ThinkSNS Plus 核心 API 版本选择的时候需要加入 v2 前缀，「圈子」拓展包为了与其他应用区别，也加入了一个特殊前缀叫做 plus-group，从而，你的完整地址前缀就是 /api/v2/plus-grouo

我们以获取所有「圈子分类」接口为例：

/api/v2/plus-group/categroies

在「圈子」API 列表中，将不会再提及 plus-group 前缀。
